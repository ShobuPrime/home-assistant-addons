# Home Assistant Add-on: Dockhand

[![Release][release-shield]][release] ![Project Stage][project-stage-shield]

Dockhand - Modern Docker management UI with real-time container orchestration and Compose stack management.

## About

Dockhand is a privacy-focused Docker management platform that provides:

- Real-time container monitoring and management
- Docker Compose stack management with Git integration
- Terminal access to containers
- Image vulnerability scanning
- Multi-environment support (local, remote, agents)
- Modern, responsive web interface

## Installation

1. Add this repository to your Home Assistant Add-on store
2. Install the "Dockhand" add-on
3. Start the add-on
4. Click "Open Web UI" or access via the sidebar
